# Specification

## Summary
**Goal:** Remove the SOS button from displaying on the home page when it should not be visible.

**Planned changes:**
- Remove or fix the conditional rendering logic that causes the SOS button/emergency section to appear unexpectedly on the home page.
- Ensure the SOS button is hidden by default and only shown when the correct display condition is explicitly met.

**User-visible outcome:** The home page loads without any unexpected SOS button or pulsing emergency section appearing.
